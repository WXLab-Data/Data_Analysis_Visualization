# 数据分析的一般流程
## 要分析什么？
数据分析第一步一定先**明确分析目标**，

明确分析目标需要清楚：

1. 读者是谁？
    - 哪一个层面的用户，教育局层面or学校层面or个体层面？
2. 读者想要看到什么？
3. 读者想通过数据分析来做什么？
